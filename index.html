FAQ


Do you use generative AI in your games?

No. I don't use any generative AI in the development of my games as I genuinely enjoy every part of game development and see no point in delegating development tasks to AI. I also believe in supporting human art and believe that every part of game development is a form of art.

Availability

When will the game be available?

The game will be available as soon as possible. Unfortunately though it's quite hard to tell when it will be ready but I aim to be done by the summer of 2027.

Where can I play the game?

The game will be made available on the Steam platform upon release. Depending on the game's success it may also be launched on additional platforms such as various consoles.

How much will the game cost?

I'm currently unsure of the exact price the game will have but I aim to keep the price relatively low to encourage a larger player base :D

Does the game require an internet connection?

No. The game does not require an internet connection.

What age rating is the game suitable for?

Features

Will the game have multiplayer?

Upon release the game will not have multiplayer support. However I would very much like to add multiplayer support in a future update.

Which languages will the game be available in?

Upon release the game will feature 3 languages, English (US), Swedish & Spanish. Later I would like to add the languages French, German, Portuguese (Brazil), Simplified Chinese & Japanese. If you would like to request another language please let me know here: Contact

Will the game have controller support?

Yes! The game will feature controller support, including (on PC) support for Sony DualShock, Sony Dualsense, Xbox One and Series X/S Controllers, Nintendo Switch Joy-Cons.

Does the game support Steam Cloud?

Yes! By default all save data is saved to Steam Cloud. (Up to 10GB per player, which is more than suitable for several LARGE worlds.)

Will the game have remappable key-binds?

Yes! There will be a settings menu for remappable key-binds.
